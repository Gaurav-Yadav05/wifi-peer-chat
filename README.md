# WiFi Peer-to-Peer Chat Application

## Features
- Real-time LAN chat
- Peer discovery
- User join notifications
- Qt GUI application

## Technologies Used
- C++
- Qt Framework
- TCP/UDP Networking

## How to Run
Open the project in Qt Creator and press the Run button.

## Description
This project demonstrates peer-to-peer communication between users connected on the same local WiFi/LAN network.
